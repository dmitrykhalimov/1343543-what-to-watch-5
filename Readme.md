#### Технологии

_`SPA`, `ES2015+`, `React`, `Redux`, `Axios`., `Jest`._

---

#### Основные команды

* Установка - `npm i`
* Запуск локального сервера - `npm run start`
* Сборка проекта без запуска локального сервера - `npm run build`
* Запуск тестирования - `npm run stylelint`
